// C언어 문제 포인터

int a=1, b=2, c=3;
int *p[3]={&a,&b,&c};
*p[0]=123;
*p[1]=456;
*p[2]=789;

printf("%d %d %d \n",a,b,c);
printf("%d %d %d \n",*p[0],*p[1],*p[2]);



//해설
a에 100번지, b에 200번지, c에 300번지 
*p[3]={100번지,200번지,300번지} 저장

*p[0]=123; 해석 => p의 0번지의 값이 123
*p[1]=456; 해석 => p의 1번지의 값이 456
*p[2]=789     => p의 2번지의 값이 789

100번지=123, 200번지=456, 300번지=789 저장 되있다.
출력문1의 a,b,c에 저장되 있는 값들 출력 (100,200,300번지)
출력문2의 *p[0]~들은 가리키고 있는 주소의 값들 출력 (100,200,300번지) 

출력해보면 정답은
123 456 789
123 456 789
